# 📱 App Navegação com Tabs

Aplicativo mobile desenvolvido em **React Native** utilizando **navegação por abas (Tab Navigation)**.
O projeto demonstra como estruturar um aplicativo com múltiplas telas e alternar entre elas utilizando um menu inferior com abas.

Esse tipo de navegação é muito utilizado em aplicativos modernos para melhorar a **experiência do usuário e organização das telas**.

---

# 🚀 Funcionalidades

* 📑 Navegação entre múltiplas telas
* 📱 Interface mobile utilizando **React Native**
* 🔄 Sistema de **Tab Navigation**
* 🧭 Estrutura básica para projetos com várias páginas
* 🎨 Interface simples e organizada

---

# 🛠️ Tecnologias utilizadas

* **React Native**
* **JavaScript**
* **Expo**
* **React Navigation**

---

# 📦 Bibliotecas utilizadas

Para instalar as dependências principais de navegação:

```bash
npm install @react-navigation/native
npm install @react-navigation/bottom-tabs
npm install react-native-screens
npm install react-native-safe-area-context
```

Se estiver usando **Expo**, execute também:

```bash
npx expo install react-native-screens react-native-safe-area-context
```

# ▶️ Como executar o projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/JoaoRamponi/appNavTelaTab.git
```

### 2️⃣ Entrar na pasta do projeto

```bash
cd appNavTelaTab
```

### 3️⃣ Instalar as dependências

```bash
npm install
```

### 4️⃣ Iniciar o projeto

```bash
npx expo start
```
